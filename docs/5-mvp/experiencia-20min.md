# MVP — Experiencia deseada (primeros 20 minutos)

> Fuente: MVP Systems Spec §3.

## Minuto 0

El jugador aparece en una playa.

> "No reconozco esta costa."
> "Tengo que mirar alrededor y entender qué puedo usar."

Ve: playa, pasto, ramas, piedras, restos de naufragio, árboles bajos, jungla espesa
bloqueando el fondo, agua, una zona donde podría empezar su campamento.

## Minuto 1-5 — Sistema de observación

```
Primer click en una rama:   "Una rama seca. Podría servirme para golpear, atar o hacer fuego."
Primer click en una piedra: "Una piedra pequeña. Es dura y cabe en mi mano."
Primer click en pasto:      "Veo fibras vegetales entre el pasto."
Primer click en jungla:     "La vegetación es demasiado cerrada. No voy a poder pasar empujando."
```

## Minuto 5-10 — Recolección física

Objetos esperados: piedra pequeña, rama seca, fibra vegetal, semilla silvestre,
restos de tela.

Aprende que el inventario es limitado (*"No tengo espacio para acomodar eso."*) y
que puede arrastrar objetos al suelo (*"Lo dejé en el suelo."*).

## Minuto 10-15 — Las manos cambian las acciones

```
Piedra en mano + roca:           "Podría intentar golpear una piedra contra otra."
Fibra en mano + rama:            "Tal vez pueda atar esto."
Piedra + rama + fibra (cerca):   "Creo que podría improvisar una herramienta."
```

Primer descubrimiento importante:

> "No es una gran herramienta, pero es mejor que mis manos."

Resultado: **herramienta rudimentaria**.

## Minuto 15-20 — Usar la herramienta y abrir el primer paso

```
Herramienta rudimentaria + árbol bajo: "Ahora puedo cortar algo más que ramas."
→ madera pobre, corteza.
Luego puede fabricar: hacha simple.
```

Primer cierre del MVP:

```
Hacha simple + jungla espesa: "Ahora sí puedo abrir un pequeño paso."
```

El jugador despeja un tile de jungla y revela una zona nueva pequeña.

> "La isla no termina acá."
