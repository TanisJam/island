# Técnicas y herramientas

> Fuente: GDD v2 §19 y §20.

## 19. Técnicas

### 19.1. Qué son

Las técnicas son **conocimiento aprendido**. No son objetos. Ej.: atadura básica,
corte rudimentario, carpintería I, molienda I, fundición I, triturado fino,
cerámica básica.

### 19.2. Niveles

Cada técnica puede tener niveles.

```
Fundición I:   permite fundir mineral común con baja eficiencia.
Fundición II:  reduce pérdidas.
Fundición III: permite enseñar a otros.
```

> ✅ **Decidido (C5):** en el MVP el conocimiento son **flags planas**
> (`idea|technique|discovery`). La transición a técnicas con nivel I/II/III queda
> planificada para más adelante. Ver
> [PREGUNTAS B1](../PREGUNTAS.md#b1--gdd-vs-mvp-modelo-completo-o-simplificado-primero).

### 19.3. Aprender por práctica

```
Usar fogata con barro → progreso en cerámica y en contención de calor.
Usar horno con mineral → progreso en fundición.
Usar piedra contra grano → progreso en molienda.
```

### 19.4. Aprender de otro jugador

Un jugador puede enseñar una técnica si: la conoce a suficiente nivel, tiene una
estación adecuada, permite enseñanza, el aprendiz trae materiales y pasa cierto
tiempo.

```
Aprender Fundición I con maestro requiere:
- visitar su base;
- usar su horno;
- traer mineral;
- traer combustible;
- 20 minutos de práctica guiada.
```

## 20. Herramientas

### 20.1. Rol de las herramientas

Las herramientas cambian el tipo y eficiencia de interacción.

```
Manos:                arrancar ramas · rebuscar · recoger.
Herramienta rudimentaria: cortar mal · raspar · romper recursos pequeños.
Hacha:                cortar árboles · obtener madera.
Pico:                 extraer piedra y mineral.
Pala:                 cavar tierra y arcilla.
```

### 20.2. Herramientas como objetos físicos

Las herramientas: ocupan espacio, tienen durabilidad, deben estar en la mano,
pueden romperse, pueden mejorarse.
