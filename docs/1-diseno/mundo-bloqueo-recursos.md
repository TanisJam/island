# Recursos del mundo, bloqueo y expansión

> Fuente: GDD v2 §15 y §16.

## 15. Recursos del mundo

### Recursos iniciales (zona inicial)

```
Ramas sueltas · Piedras pequeñas · Pasto alto · Semillas silvestres ·
Fibra vegetal · Restos de naufragio · Árboles bajos · Rocas pequeñas ·
Agua · Arena · Tierra
```

### Interacciones iniciales (resumen por recurso y manos)

**Árbol:** manos → arrancar ramas; piedra → golpear/raspar; herramienta
rudimentaria → cortar madera pobre; hacha → obtener madera.

**Pasto:** manos → rebuscar/arrancar fibra/encontrar semillas; herramienta cortante
→ cortar pasto/obtener más fibra.

**Roca:** manos → examinar; piedra → golpear (fragmentos, bajo rendimiento);
pico → extraer piedra y mineral.

**Suelo:** manos → rebuscar (piedra pequeña, semilla o arcilla rara); pala → cavar
tierra, arcilla, raíces.

**Agua:** manos → beber/mojar objeto; recipiente → recolectar agua.

## 16. Bloqueo y expansión de la isla

### Zona inicial

El jugador comienza en una pequeña zona de playa/bosque. Detrás hay jungla densa.

> "La vegetación es demasiado espesa. No puedo avanzar así."

### Formas de abrir caminos

```
Jungla densa:  herramienta cortante + energía + tiempo (quizá fuego controlado futuro).
Rocas grandes: pico + técnica de extracción + explosivos o máquinas futuras.
Río:           balsa / puente / cuerda + conocimiento de carpintería.
Acantilado:    cuerda + escalera + anclaje.
Oscuridad/cueva: antorcha + lámpara + técnica de exploración.
```

### Progresión orgánica

Cada nuevo bloqueo debe sugerir un objetivo.

> "Si pudiera cortar estos árboles, quizás encontraría un paso."
> "El agua no parece profunda, pero no voy a cruzar cargando todo encima."
