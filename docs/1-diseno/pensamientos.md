# Sistema de pensamientos

> Fuente: GDD v2 §9. Estructura del tipo en [modelo de datos](../2-dominio/modelo-de-datos.md#pensamientos);
> detalle MVP en [pensamientos MVP](../5-mvp/pensamientos.md).

Todos los mensajes deben escribirse en **primera persona**. Nunca mensajes técnicos.

| No decir | Decir |
|---|---|
| `Inventory full.` | "No tengo espacio para acomodar eso." |
| `Requires sharp tool.` | "Necesito algo con filo para cortar esto." |
| `Research +2.` | "Empiezo a entender cómo contener mejor el calor." |

## 9.1. Tipos de pensamiento

```
Observación:
"Veo pasto alto. Podría esconder semillas."

Idea:
"Quizás pueda atar esta piedra a una rama."

Descubrimiento:
"Ahora entiendo algo: el barro cambia con el calor."

Advertencia:
"Ese bosque es demasiado espeso para atravesarlo así."

Fallo útil:
"No funcionó, pero aprendí que la madera se quema demasiado rápido."

Recuerdo:
"Desperté en esta playa después de la tormenta."
```

## 9.2. Historial

El historial de pensamientos sirve como **diario emergente** del jugador.

Filtros futuros: todo, ideas, descubrimientos, peligro, investigación, social.
